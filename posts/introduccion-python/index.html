Introducción a Python para Principiantes

29 de enero de 2025

¿Qué es Python?

Python es un lenguaje de programación de alto nivel, interpretado y de propósito general. Es conocido por su sintaxis clara y legible, lo que lo hace perfecto para principiantes.

Instalación de Python

En Linux

sudo apt update
sudo apt install python3 python3-pip

Verificar la instalación

python3 --version

Tu primer programa

Crea un archivo llamado hola.py:

print("¡Hola, mundo!")

Ejecútalo:

python3 hola.py

Variables en Python

nombre = "José María"
edad = 25
altura = 1.75

print(f"Me llamo {nombre}, tengo {edad} años y mido {altura}m")

Conclusión

Python es un excelente lenguaje para comenzar a programar. En próximos artículos veremos estructuras de datos, funciones y más.
