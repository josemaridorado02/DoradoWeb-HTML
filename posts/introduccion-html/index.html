    Introducción a HTML5#

    HTML5 es la última versión del lenguaje de marcado utilizado para crear páginas web.

    ¿Qué es HTML?#

    HTML (HyperText Markup Language) es un lenguaje de marcado que define la estructura del contenido web.

    Elementos básicos#

    Los elementos HTML más comunes son:

    • <h1> a <h6> - Títulos
    • <p> - Párrafos
    • <a> - Enlaces
    • <img> - Imágenes
    • <ul>, <ol> - Listas

    Estructura básica#

    <!DOCTYPE html>
<html>
<head>
    <title>Mi página</title>
</head>
<body>
    <h1>Hola mundo</h1>
</body>
</html>
